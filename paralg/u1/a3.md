# CRCW
## OR
```C
bool or(bool x[n]) {
	bool result = false
	for each i in {0,...,n-1} do in parallel
		if(x[i]) result = true
	return result
}
```

## AND
```C
bool and(bool x[n]) {
	bool result = true
	for each i in {0,...,n-1} do in parallel
		if(!x[i]) result = false
	return result
}
```
Θ(1)
# EREW
## OR
```C
bool or(bool x[n]) {
	bool tmp[0:n-1], xcopy[0:n-1]
	for each i in {0,...,n-1} do in parallel
		xcopy[i] = x[i]
	for k = 0 to log2(n-1) do
		for	each i in {0,...,n-1} do in parallel {
			tmp[i] = x[(i + 2 ^ k) % n]
			if(!x[i]) x[i] = tmp[i]
		}
	return x[0]
}
```
Θ(log2(n))
